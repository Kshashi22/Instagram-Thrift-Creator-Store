# Instagram Thrift Creator Store

## Project Overview

The **Instagram Thrift Creator Store** is an online thrift shopping platform designed for creators, influencers, and small businesses to sell pre-owned, vintage, and trendy fashion products directly through social media-driven engagement.

This project focuses on building a modern thrift store experience inspired by Instagram-style product showcasing, where users can browse fashion collections, explore creator profiles, and purchase affordable second-hand products in a user-friendly interface.

The platform aims to support:

* Sustainable fashion
* Small thrift businesses
* Creator-based selling
* Affordable shopping
* Social commerce experiences

---

# Objectives

* Create a creator-focused thrift shopping platform
* Promote sustainable and reusable fashion
* Provide an Instagram-inspired UI/UX
* Enable product listing and order management
* Support secure and smooth shopping experiences

# Installation and Setup

## Clone the Repository

```bash id="4yjj33"
git clone https://github.com/your-username/instagram-thrift-creator-store.git
```

## Navigate to Project Folder

```bash id="thn0sw"
cd instagram-thrift-creator-store
```

## Install Dependencies

```bash id="jlwmfc"
npm install
```

## Run Frontend

```bash id="o7d5xe"
npm start
```

## Run Backend

```bash id="b0js5n"
npm run server
```
